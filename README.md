# PyChat

PyChat is a lightweight, open-source chat application designed to allow anyone to connect and communicate on a global scale. It provides a simple and accessible platform for users around the world to chat in real-time.

## 🔍 Troubleshooting & First Launch Note

### Stuck on Update during First Launch?
If you open the application for the first time (or after an update) and the launcher appears to be **stuck or frozen** on the update screen, simply **close and reopen the app**.

* **Why does this happen?** On the first launch, the application is actively downloading, verifying, or updating core system files in the background. Depending on your network or system permissions, the interface might occasionally stop responding visually.
* **Is it safe to close?** **Yes.** Closing the application while it is stuck **will not corrupt any files** or damage the installation. Reopening it will safely resume the process or start the application normally.
